# Contact_Book
книга контактов нужна для хранения контактов пользователя. Приложение использует структуру данных  списки

возможности:
    - добавление элемента
    - удаление элемента
    - просмотр списка

Приложение работает в интерактивном режиме.
Каждый элемент состоит из:
    - ФИО
    - ТЕЛЕФОН
    - EMAIL
    - АДРЕС
